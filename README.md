# Open-Library [ALPHA]
An FOSS library desktop app much like Amazon Kindle, where people can read books and even add books.

## About

Open Library is created by Rangerdevv. It is meant to be an offline desktop reading platform where users can read books on any topic their heart desires. It is meant to be simple and feel like the user is in an actual library. 

The goal of Open Library is to be free of any bloat, distractions, and most importantly be extremly user friendly.
